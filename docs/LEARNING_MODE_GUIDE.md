# 学习模式使用指南

**版本**: 1.0.0  
**最后更新**: 2026-03-28

---

## 📖 概述

学习模式是 Scriptor 的核心功能之一，它允许 AI 像"新员工"一样从对话中积极学习知识，并通过确认机制确保知识的准确性。学习完成后，AI 可以切换到"授课模式"，作为权威专家为学生解答问题。

### 核心理念

- **学习模式**: AI 积极提取知识，但需要导师确认
- **授课模式**: AI 作为专家，知识库锁定为只读
- **日常模式**: 正常工作状态

---

## 🎯 功能特性

### 1. 三态认知模型

| 状态 | 图标 | 特点 | 使用场景 |
|------|------|------|---------|
| **日常模式** | 🟢 | 正常工作，知识可读写 | 日常对话、任务处理 |
| **学习模式** | 🟡 | 积极提取，待确认机制 | 学习新知识、业务培训 |
| **授课模式** | 🔴 | 只读锁定，权威专家 | 教学、答疑、知识输出 |

### 2. 待确认知识机制

学习模式下提取的知识不会立即写入数据库，而是进入"待确认区"，等待导师确认后才正式成为知识库的一部分。

**优势**:
- ✅ 防止错误知识污染数据库
- ✅ 允许导师修正和完善
- ✅ 提高知识质量

### 3. 双轨写入

确认后的知识会同时写入:
- **知识库 (Knowledge Base)**: 向量数据库，支持语义搜索
- **知识图谱 (Knowledge Graph)**: 图结构，支持关系推理

## 🚀 快速开始

### 1. 启动学习模式

```
/开始学习
```

**效果**:
- AI 进入学习状态
- 系统提示"🟡 已切换到学习模式"
- AI 开始积极从对话中提取知识

### 2. 进行教学对话

在学习模式下，与 AI 进行教学对话，例如：

```
用户：办理刑事案件的基本流程是什么？
AI: 让我记录一下...（自动提取知识点）

用户：侦查阶段需要注意什么？
AI: 好的，我会把这个要点也记下来。
```

### 3. 查看待确认知识

```
/学习状态
```

**输出示例**:
```
📊 学习状态详情

当前状态：🟡 学习模式
会话 ID: user_12345
待确认知识：3 条
已确认知识：15 条

待确认列表:
1. 刑事案件办理流程 (fact)
2. 侦查阶段注意事项 (skill)
3. 证据收集规范 (rule)
```

### 4. 确认知识

```
/确认知识
```

**效果**:
- 所有待确认知识写入知识库和知识图谱
- 待确认列表清空
- 已确认知识计数 +1

### 5. 结束学习模式

```
/结束学习
```

**效果**:
- 切换回日常模式 🟢
- 未确认的知识保留在待确认区
- 下次学习时可继续确认

## 📚 高级功能

### 授课模式

当你需要 AI 作为专家进行教学时：

```
/开始授课
```

**特点**:
- 🔴 知识库锁定为只读
- AI 以权威专家身份回答
- 不允许修改知识库
- 适合教学、答疑场景

结束授课：
```
/结束授课
```

### 知识修改

如果发现提取的知识有误，可以在确认前要求修改：

```
用户：修改第 2 条知识的标题为"侦查阶段核心要点"
AI: 好的，已修改。
```

## ⚙️ 配置说明

在 `_conf_schema.json` 或 AstrBot 配置界面中调整学习模式参数：

### learning_mode_enabled
- **类型**: bool
- **默认**: true
- **说明**: 是否启用学习模式功能

### learning_max_pending_items
- **类型**: int
- **范围**: 1-100
- **默认**: 10
- **说明**: 每个会话最大待确认知识数量

### teaching_mode_enabled
- **类型**: bool
- **默认**: true
- **说明**: 是否启用授课模式功能

### teaching_strict_mode
- **类型**: bool
- **默认**: true
- **说明**: 授课模式下是否严格禁止知识库修改
  - `true`: 硬性拦截，完全不允许修改
  - `false`: 仅警告，但仍允许修改

## 🐛 常见问题

### Q1: 为什么我提取的知识没有写入数据库？

**A**: 学习模式下，知识会先进入"待确认区"，需要执行 `/确认知识` 才会正式写入。

### Q2: 待确认知识最多能存多少条？

**A**: 默认 10 条，可通过 `learning_max_pending_items` 配置调整。

### Q3: 授课模式下真的完全不能修改知识吗？

**A**: 
- `teaching_strict_mode=true`: 硬性拦截，无法修改
- `teaching_strict_mode=false`: 会警告，但仍允许修改

### Q4: 学习模式会一直开启吗？

**A**: 不会。学习模式需要手动开启，也可以手动关闭。建议学习完成后及时 `/结束学习`。

### Q5: 待确认知识会过期吗？

**A**: 不会自动过期，但会话结束后（如重启 AstrBot），待确认知识可能会丢失。建议及时确认。

## 📊 技术架构

### LearningManager 核心类

```python
class LearningManager:
    - CognitiveState: 三态认知模型
    - KnowledgeExtraction: 知识提取
    - LearningSession: 学习会话
    
    核心方法:
    - get_session(): 获取会话
    - set_state(): 切换状态
    - add_pending_knowledge(): 添加待确认知识
    - confirm_pending_knowledge(): 确认知识（双轨写入）
    - revise_pending_knowledge(): 修改知识
    - get_state_prompt_suffix(): 动态 Prompt 注入
```

### 双轨写入流程

```
用户确认知识
    ↓
LearningManager.confirm_pending_knowledge()
    ↓
┌─────────────────────────────────┐
│  遍历待确认知识列表              │
└─────────────────────────────────┘
    ↓
┌──────────────┐    ┌──────────────┐
│  写入知识库   │    │  写入知识图谱 │
│  (ChromaDB)  │    │  (JSON File) │
└──────────────┘    └──────────────┘
    ↓                     ↓
┌─────────────────────────────────┐
│  记录日志 + 错误处理             │
└─────────────────────────────────┘
    ↓
┌─────────────────────────────────┐
│  清空待确认列表                  │
│  更新已确认计数                  │
└─────────────────────────────────┘
```

## 🎓 应用场景案例

### 案例 1: 新员工培训

**场景**: 公司 HR 培训 AI 了解公司制度

```
/开始学习

HR: 我们公司的工作时间是早上 9 点到下午 6 点，午休 1 小时。
AI: 好的，已记录"公司工作时间"知识点。

HR: 请假流程需要先在 OA 系统提交，然后直属领导审批。
AI: 明白，已记录"请假流程"知识点。

HR: 还有，报销需要在每月 25 号前提交。
AI: 好的，"报销截止时间"也已记录。

/学习状态
# 查看已提取 3 条知识

/确认知识
# 写入数据库

/结束学习
```

### 案例 2: 专业知识教学

**场景**: 法律专家培训 AI 了解刑事诉讼法

```
/开始学习

专家：刑事诉讼分为三个阶段：侦查、审查起诉、审判。
AI: 已记录"刑事诉讼阶段"知识点。

专家：侦查阶段由公安机关负责，期限为 2 个月。
AI: 好的，"侦查期限"知识点已记录。

专家：审查起诉由检察院负责，期限为 1 个月。
AI: 明白，"审查起诉期限"知识点已记录。

/学习状态
# 查看待确认列表

专家：等等，侦查期限可以延长，你记录的不准确。
AI: 好的，请告诉我正确的内容。

专家：侦查期限一般为 2 个月，但经批准可以延长。
AI: 好的，已修正知识点。

/确认知识
/结束学习
```

### 案例 3: 授课模式答疑

**场景**: AI 作为法律专家为学生答疑

```
/开始授课

学生：老师，刑事案件侦查期限是多久？
AI: 根据我国刑事诉讼法，侦查期限一般为 2 个月，
    但经上一级检察院批准可以延长。

学生：什么情况下可以延长？
AI: 对于案情复杂、期限届满不能终结的案件，
    可以延长 1 个月。

学生：这个知识点在你的知识库里吗？
AI: 是的，这是我之前系统学习过的内容。

/结束授课
```
