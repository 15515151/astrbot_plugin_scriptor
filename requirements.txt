# 🔧 故障排查指南
# ============================================
#
# Q1: 出现 "Microsoft Visual C++ 14.0 or greater is required" 错误？
# A: 说明你在 Python 3.13+ 上尝试编译 C++ 扩展。
#    解决方案：
#    • Windows: 运行 install.bat（自动用 Conda）
#    • 或安装 Miniconda: https://docs.conda.io/en/latest/miniconda.html
#
# Q2: chromadb 安装成功但 import 失败？
# A: 可能是 hnswlib.dll 缺失。尝试：
#    conda install -c conda-forge chroma-hnswlib --force-reinstall
#
# Q3: 想快速测试，不想安装全部依赖？
# A: 使用核心依赖文件：
#    pip install -r requirements-core.txt
#    （不包含向量数据库功能）
#
# Q4: Linux/Mac 用户？
# C++ 扩展通常更容易编译。直接：
#    pip install -r requirements.txt
#    如果失败，安装 build-essential (Ubuntu) 或 Xcode Command Line Tools (Mac)
